# Statistic Service

## Overview
Statistic Service aggregates board and workspace data and exposes it via a REST API.

## Docs
- Swagger UI: `{statistic_service_url}/docs`
- OpenAPI JSON: `{statistic_service_url}/docs.json`

## Development
Run the service (from this folder):

```bash
pnpm install
pnpm dev
```

## Tests
```bash
pnpm test
```
